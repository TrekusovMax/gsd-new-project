## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Удалён cacheControlMaxAge из клиентского вызова upload()**
- **Found during:** Task 1 — TypeScript verification
- **Issue:** RESEARCH.md и PLAN описывают `cacheControlMaxAge: 3600` в вызове `upload()` из `@vercel/blob/client`, но этот параметр отсутствует в `UploadOptions` клиентского SDK (версия 2.3.3). Он доступен только на сервере через `onBeforeGenerateToken`.
- **Fix:** Удалён `cacheControlMaxAge` из client-side `upload()` вызова. INFRA-02 TTL compliance сохраняется — параметр уже установлен в `/api/upload/route.ts` (реализовано в PLAN-1.3).
- **Files modified:** features/pdf/usePdfUpload.ts
- **Commit:** 8dac696

**2. [Rule 1 - Bug] Исправлен тип RefObject для React 19**
- **Found during:** Task 2 — TypeScript verification
- **Issue:** В React 19 `useRef<HTMLInputElement>(null)` возвращает `RefObject<HTMLInputElement | null>`, но prop в DropZone был объявлен как `RefObject<HTMLInputElement>` — TypeScript strict ошибка.
- **Fix:** Обновлён тип prop `fileInputRef` в DropZone до `React.RefObject<HTMLInputElement | null>`.
- **Files modified:** components/DropZone.tsx
- **Commit:** 1cad99a

**3. [Rule 1 - Bug] ARIA-атрибуты вынесены из Ant Design Progress в обёртку**
- **Found during:** Task 2 — TypeScript verification
- **Issue:** Ant Design `Progress` компонент не принимает `role`, `aria-valuenow`, `aria-valuemin`, `aria-valuemax`, `aria-label` как props — TypeScript strict ошибка "Property 'role' does not exist on type 'ProgressProps'".
- **Fix:** Обёрнут `<Progress>` в `<div>` с ARIA-атрибутами.
- **Files modified:** components/UploadProgress.tsx
- **Commit:** 1cad99a
