## Decisions Made

- Used `LiteralObject` directly in `context.stream(newImageBytes, newStreamDict)` instead of going through `context.obj({...})` — the latter returns `PDFDict` which is incompatible with `context.stream`'s second parameter type
- SSRF guard uses regex `/\.blob\.vercel-storage\.com$/` on `new URL(blobUrl).hostname` — rejects any non-Vercel-Blob URL before server-side fetch
- `error.message.toLowerCase()` for encrypted PDF detection makes the check case-insensitive (pdf-lib message format may vary)
